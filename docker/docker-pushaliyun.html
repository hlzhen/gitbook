2.阿里云镜像仓库操作指南

1. 登录阿里云Docker Registry
- docker login --username=阿里云账号名称 registry.cn-hangzhou.aliyuncs.com
- 用于登录的用户名为阿里云账号全名，密码为开通服务时设置的密码。

2. 从Registry中拉取镜像
- docker pull registry.cn-hangzhou.aliyuncs.com/docker-repository-01/docker-dev1:[镜像版本号]

3. 将镜像推送到Registry
- docker login --username=超人不会飞icu registry.cn-hangzhou.aliyuncs.com
- docker tag [ImageId] registry.cn-hangzhou.aliyuncs.com/docker-repository-01/docker-dev1:[镜像版本号]
- docker push registry.cn-hangzhou.aliyuncs.com/docker-repository-01/docker-dev1:[镜像版本号]
